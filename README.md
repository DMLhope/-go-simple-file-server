# -go-simple-file-server
用go实现的简单的文件服务器
```shell
$ go run main.go -h                                                          
  -addr string
        ip地址,默认为127.0.0.1 (default "127.0.0.1")
  -h    帮助
  -p int
        端口号,默认为9000 (default 9000)
  -path string
        路径,默认为./当前路径 (default "./")
```
